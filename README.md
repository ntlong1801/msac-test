### Instructions for running the project

1. Prerequisites:
    - Node.js 
    - npm or yarn
    - Git

2. Installation:
    ```bash
    git clone [repository-url]
    cd [project-directory]
    npm install
    ```

3. Configuration:
    - Copy `.env.example` to `.env`
    ```bash
        cp .env.example .env
    ```
    - Update environment variables

4. Development:
    ```bash
    npm run dev
    ```

5. Production:
    ```bash
    npm run build
    npm start
    ```

6. Postman:
    - Import ToDoList.postman_collection.json to Post man
    - Testing Api

7. Testing:
    ```bash
    npm test
    ```
